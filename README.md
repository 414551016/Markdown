# 字型
- 基礎字體樣式
```
粗體：在文字前後加上兩個星號 **粗體**
斜體：在文字前後加上一個星號 *斜體*
斜粗體：在文字前後加上三個星號 ***斜粗體***
刪除線：在文字前後加上兩個波浪號 ~~刪除線~~
內嵌程式碼：在文字前後加上反引號 `內嵌程式碼`
```
- 字體顏色與大小
```
若需變更顏色或大小，可利用內嵌 HTML 語法達成：
變更顏色：<span style="color:red;">紅色文字</span>
變更大小：<span style="font-size:24px;">放大文字</span>
變更字型（依裝置支援度而定）：<span style="font-family:Arial;">特定字型</span>
```

# 水平線
在 Markdown 中建立水平線非常簡單。只需在獨立的一行中輸入 三個或以上 的星號 (*)、破折號 (-) 或底線 (_) 即可。
支援的語法格式您可以使用以下任一種方式來繪製分隔線：
```
---
***
___
```
- 執行結果：
---
***
___
注意事項
- 獨立一行：確保這三個符號是單獨佔據一行，前後不要包含其他文字。
- 留白：在水平線的上方與下方各空一行，排版效果會更好、閱讀起來更清晰。
